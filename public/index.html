Research

My research aims to assess ChatGPT's performance in different domains, such as History, Social Science, and Computer Security. By leveraging MongoDB for data storage and Node.js with Express.js for server-side interactions, you're able to evaluate the accuracy and response times of ChatGPT's answers. The project integrates the ChatGPT API to automatically retrieve responses to pre-set questions, allowing you to measure performance and compare it to expected answers. This will provide insights into ChatGPT's efficiency, accuracy, and potential areas for improvement.

ChatGPT Efficiency Evaluation

Research Approach

This evaluation uses Node.js for server-side logic, MongoDB as the database, and the ChatGPT API for responses. The project analyzes response times and accuracy across various topics.

Dataset Overview

The dataset includes questions from History, Social Science, and Computer Security, with each domain containing 50 questions. These evaluations aim to measure how well ChatGPT performs in these areas.
